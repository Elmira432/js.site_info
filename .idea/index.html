Alles Wichtige über JavaScript

Die Informationen stammen aus Wikipedia.

1

Was ist JavaScript?

JavaScript ist eine Programmiersprache, die hauptsächlich für Webseiten benutzt wird. Mit ihr kann man eine Seite interaktiv machen – also zum Beispiel auf Klicks reagieren, Inhalte verändern oder kleine Spiele bauen. Sie läuft direkt im Browser, ohne dass man etwas installieren muss.

2

Wer hat JavaScript erfunden?

JavaScript wurde von Brendan Eich erfunden. Er hat die Sprache im Jahr 1995 entwickelt, als er bei der Firma Netscape gearbeitet hat. Angeblich hat er nur zehn Tage gebraucht, um die erste Version zu schreiben.

3

Wann ist sie entstanden?

JavaScript wurde 1995 zum ersten Mal veröffentlicht. Damals hieß sie kurz „Mocha“, dann „LiveScript“ und schließlich „JavaScript“. Der Name wurde gewählt, weil die Sprache Java damals sehr beliebt war – obwohl die beiden Sprachen kaum etwas miteinander zu tun haben.

4

Wo wird sie benutzt?

Fast jede Webseite benutzt heute JavaScript: zum Beispiel Google, YouTube, Instagram oder TikTok. Außerdem kann man mit JavaScript auch Apps für das Handy machen, Server programmieren oder sogar kleine Roboter steuern.

5

Wie sieht JavaScript aus?

JavaScript-Code besteht aus Anweisungen, die der Computer der Reihe nach ausführt. Hier ein kleines Beispiel: Wir speichern einen Namen in einer Variablen und geben dann eine Begrüßung aus.

// Begrüßung
const name = "Paula";
console.log("Hallo " + name + "!");
6

Warum ist sie so wichtig?

JavaScript ist eine der am meisten benutzten Programmiersprachen der Welt. Wer Webseiten bauen oder im Internet etwas programmieren möchte, kommt fast nicht an ihr vorbei. Auch viele Berufe in der IT setzen sie voraus.
